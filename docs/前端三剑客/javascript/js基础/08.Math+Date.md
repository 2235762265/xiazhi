# Math+Date

## 一、Math数学对象

> Math 是 js 的一个内置对象，提供了一堆的方法帮助我们操作 **数字**

### Math对象的方法

```js
c	Math.ceil(num)  	//对数字进行向上取整
	Math.floor(num)  	//对数字进行向下取整
	Math.round(num)  	//对数字进行四舍五入
	Math.pow(num,n)  	//计算数字num的n次方
	Math.PI  			//π 圆周率
	Math.abs(num)  		//绝对值
	Math.max(x,y)  		//返回最大值
	Math.min(x,y)  		//返回最小值
	Math.sqrt(num)      //返回num的平方根
	Math.random() 		//返回介于 0 ~ 1 之间的一个随机数
```

> 自定义区间的随机数公式： `Math.floor( Math.random()*(max-min+1)+min )`

## 二、Date 日期对象!!!

### 简介

> 时间对象也是JS中使用频率非常高的对象，页面中很多的效果也是依赖时间对象来完成的。
> Date对象是 JavaScript 原生的时间库。它以1970年1月1日00:00:00作为时间的零点，可以表示的时间范围是前后各1亿天（单位为毫秒）。



### 日期对象的创建

> 创建日期对象需要用到Date()构造函数。

```js
    // 返回当前用户客户端时间
    var date1 = new Date();

    // 设置规定的时间
    var date2 = new Date('2019-04-23 14:20:00');

    // 设置规定的时间
    // 按照顺序分别表示：年，月，日，时，分，秒，毫秒
    var date3 = new Date(2019,4,23,14,20,0,0);
```

> 上面的代码就是创建了三个时间对象，第二个和第三个都是设置指定的时间，但是它们的用法却是区别很大，不仅仅是语法上的区别，在结果上也是有区别的，结果如下：

```js
    console.log(date1);// 当前时间
    console.log(date2);// Tue Apr 23 2019 14:20:00 GMT+0800 (中国标准时间)
    console.log(date3);// Thu May 23 2019 14:20:00 GMT+0800 (中国标准时间)
```

> 从结果中我们很明显的发现了第二个时间对象和第三个时间对象我们输入的参数都是4，但是结果却是一个四月，一个五月。
> 在第三种创建方式中数字**0**代表一月，数字11代表十二月，所以现在明白了为什么我们明明写了4结果却是五月了。
> 所以建议大家在创建一个指定的时间对象的时候使用第二种方式。



### 日期对象的方法

> 我们在页面中得到一个时间对象，但是这个时间对象是不能直接使用的(毕竟不是每个人都能够看懂时间对象的语法)。
> 所以我们需要将时间对象转换成一个普通人就能够看得懂的样式。

> 我们可以通过如下方法进行转换：

- getFullYear():以四位数字返回年份
- getMonth():返回月份(0~11)，0表示1月，以此类推  (注：写代码时对应加1)
- getDate():返回一个月中的某一天(1~31) 
- getHours():返回小时 (0 ~ 23)
- getMinutes():返回分钟 (0 ~ 59)
- getSeconds():返回秒数 (0 ~ 59) 
- getDay():返回一周中的某一天(0~6)，0表示周日 
- getTime():返回1970年1月1号至今的毫秒数 - 时间戳            1秒 === 1000毫秒
