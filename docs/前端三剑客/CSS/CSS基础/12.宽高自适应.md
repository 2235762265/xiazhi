# 宽高自适应

## 概念

网页布局中经常要定义元素的宽和高。但很多时候我们希望元素的大小能够根据窗口或子元素自动

调整，这就是自适应

元素自适应在网页布局中非常重要，它能够使网页显示更灵活，可以适应在不同设备、不同窗口和

不同分辨率下显示。

## 使用

宽度自适应

元素宽度设置为100%。（块元素宽度默认为100%） width: 100%;

高度自适应

元素{height: auto;} 高度不写

场景：

两栏，三栏，上下两栏，上下三栏



通常所用属性

width:calc(100% - 100px);          （-号两边用空格）

意为宽度为父元素宽度-像素值

高度同理
