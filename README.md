# `Vuepress`博客源码

### 运行

```
# 打开cmd 全局安装yarn
$ npm i -g yarn

# 设置yarn淘宝源，使用npm的同学，也是可以设置淘宝源
$ yarn config set registry https://registry.npm.taobao.org

$ git clone https://gitee.com/xarzhi/xarzhi.git

# 打开CMD命令行窗口
$ cd xarzhi

# 安装vuepress
$ yarn add -D vuepress 或 npm install -g vuepress

# 运行
$ yarn dev  或 npm run dev

# 打包
$ yarn build
```
